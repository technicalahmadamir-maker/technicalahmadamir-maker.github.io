## 🔧 GitHub Par Upload Karna

### Step 1 – GitHub Account Banayein (agar nahi hai)
👉 https://github.com/signup

### Step 2 – New Repository Banayein
1. GitHub open karein
2. Click karein **"New"** (top left plus icon ya green button)
3. Repository name dein: `ahmad-amir-portfolio`
4. **Public** select karein
5. **"Create repository"** click karein

### Step 3 – Files Upload Karein
1. New repo mein click karein **"uploading an existing file"**
2. Saari files drag & drop karein (`index.html`, `style.css`, `script.js`, `vercel.json`)
3. **"Commit changes"** click karein

---

## ⚡ Vercel Par Deploy Karna

### Method 1 – GitHub se (Recommended)
1. 👉 https://vercel.com/signup par jaayein
2. **"Continue with GitHub"** click karein
3. **"New Project"** click karein
4. Apni `ahmad-amir-portfolio` repository select karein
5. **"Deploy"** click karein ✅
6. 2 minute mein aapki site live hogi!

### Method 2 – Direct Upload
1. 👉 https://vercel.com/new jaayein
2. **"Deploy without Git"** → folder drag & drop karein
3. **"Deploy"** click karein ✅

## ✏️ Customize Karna

### Apna Photo Add Karna
- `index.html` mein `avatar-initials` div dhundhein aur image tag se replace karein

### Projects Update Karna
- `index.html` mein `#projects` section mein apne real projects add karein

### Social Media Links Add Karna
- Footer mein apne LinkedIn, Instagram, Fiverr links add karein

### Custom Domain Lagana
- Vercel Dashboard → Project → Settings → Domains
- Apna domain add karein (e.g., `www.ahmadamir.com`)
